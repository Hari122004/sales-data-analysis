# 🛒 Sales Data Analysis Dashboard

## 📌 Overview
This project is an interactive Sales Data Analysis Dashboard built using Streamlit, Pandas, NumPy, and Matplotlib. It allows users to upload an e-commerce dataset and analyze sales performance in real time.

---

## 🚀 Features
- Upload CSV dataset
- Data cleaning and preprocessing
- Total sales calculation
- Top-selling products analysis
- Category-wise revenue insights
- City-wise sales analysis
- Customer spending analysis
- Interactive charts (bar, pie, histogram)

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
